# allroads_public_assets

## Included Data

- data/overture-tiles/
  - Processed tile data derived from Overture Maps
  - road/ (road tiles)
  - poi/ (POI tiles)
  - manifest.json (generation metadata)
- data/N06-20/
  - N06-20_Joint.geojson (derived from Japan National Land Numerical Information)
